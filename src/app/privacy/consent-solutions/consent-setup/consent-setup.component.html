Implementing Consent Preference

Note: To provide the network path between the Adzapier Portal and your domain, you must insert script before closing the HEAD tag on the web page of your domain.

Setup Script

//Copy and paste this script into header of your web page. It is important that this script must be the first script on your web page
{{addScript}} {{scriptUrl}} {{closeScript}}

Initialization (init Configuration)

//Copy this script and paste it below the setup script on your web page.The AppID and PropID with this script are automatically generated.
window.CP_SDK_ADZAPIER.init
